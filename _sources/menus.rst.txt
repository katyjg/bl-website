Menus
=====

To add items to the main menu, from the main admin sidebar, select "Settings" >> "Main menu".

.. image:: images/menu-edit.png
   :align: center
   :alt: Main Menu Editing

For a page to be displayed on the main menu, "Show in menus" must be selected under its "Promote" tab.

.. image:: images/show-in-menu.png
   :align: center
   :alt: Show in Menus
