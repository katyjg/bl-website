  • CMCF Beamline Website

This website template is built with the Django framework and Wagtail CMS. Features and functionality are meant to address the needs of any synchrotron beamline public website, and were developed specifically for use at the Canadian Macromolecular Crystallography Facility beamlines at the Canadian Light Source.

Source code can be found at https://github.com/katyjg/bl-website.

Note

Find more information about: - Django: https://docs.djangoproject.com/en/3.0/ - Wagtail: https://docs.wagtail.io/en/stable/
